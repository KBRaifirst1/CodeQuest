# CodeQuest

A beginner coding-learning app with classes, an AI teacher, projects, and cloud save (Supabase).

## Deploy
1. Run `supabase-schema.sql` in your Supabase project's SQL Editor.
2. Add env vars `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` (locally in `.env`, and in Vercel).
3. Import this repo in Vercel → it auto-builds (Vite) and deploys.

See MIGRATION-GUIDE.md for full step-by-step.
